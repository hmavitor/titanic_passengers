No contexto imigratrio do Titanic, a classe social se relaciona com nmero de familiares bordo, que tambm parece ter afetado a chance de sobrevivncia ao naufrgio.

V. Limitações e Transformações¶

O conjunto de dados importado apresentou rtulos com iniciais maisculas, e foram renomeados de acordo com as convenes apresentadas no curso. Os dados de idade, cabine e embarcao apresentaram valores ausentes/nulos. Nenhuma anlise foi realizada com os dados de cabine, por conta do excesso de NaNs (687). Assim, essa coluna foi descartada com drop(), o que limitou a anlise dos dados dessa perspectiva das cabines. J com os dados de idade e embarcao, a quantidade de NaNs foi menor (177 e 2, respectivamente). Assim, as linhas foram descartadas com dropna(), o que diminuiu a amostragem total e pode ter afetado a representatividade dos dados.

Na ltima pergunta, a anlise no considera que famlias diferentes podem ter o mesmo sobrenome, e que muitos passageiros sobreviventes eram os nicos representantes de suas familias, no compondo uma familia "real", no sentido coletivo.

No houve necessidade de transformao dos tipos de dados.

# Cria 2 DataFrames para diferentes anlises com os valores nicos de sobrevivncia
df_survived = df.query('survived == 1')
df_nsurvived = df.query('survived == 0')

# Cria a funo prop()

def prop(column, surv):
    """ Calcula as propores dos valores nicos de uma varivel dos passageiros - 
    sobreviventes ou no sobreviventes - do Titanic.
    Entrada:
    column = rtulo da varivel (string)
    surv = 1 para dados de sobreviventes; 0 para no sobreviventes (int)
             
    Sada: propores de cada valor nico de column, apenas das linhas de passageiros definidos em surv (DataFrame) """
    
    surv_count = df.query('survived == @surv')[column].value_counts()
    surv_size = len(df.query('survived == @surv'))
    
    return surv_count/surv_size
